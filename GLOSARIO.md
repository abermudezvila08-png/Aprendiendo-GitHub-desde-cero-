# Glosario del Proyecto

## Archivo

Un documento.

Puede contener texto, imágenes o código.

## Carpeta

Lugar donde se guardan archivos relacionados.

## Commit

Una fotografía del estado del proyecto.

Cada commit guarda una versión para poder volver atrás cuando sea necesario.

## Rama

Una copia del proyecto para experimentar sin romper el original.

## README

La portada del proyecto.

Es lo primero que verá cualquier visitante.

## Issue

Una tarea pendiente.

Ejemplo:

"Hacer el storyboard."

## Release

Una versión publicada del proyecto.
