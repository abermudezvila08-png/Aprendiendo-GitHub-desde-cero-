# Lección 001

## ¿Qué es Git?

Git es una máquina del tiempo.

No guarda únicamente archivos.

Guarda la historia de los archivos.

Cada vez que hacemos un cambio importante tomamos una fotografía.

Esa fotografía recibe un nombre.

Ese nombre es el mensaje del commit.

---

## Ejemplo

Hoy escribimos un README.

Mañana agregamos un glosario.

Pasado mañana cometemos un error.

Gracias a Git podemos volver al momento en que todo funcionaba.

Eso convierte a Git en una máquina del tiempo para proyectos.

---

## Ejercicio

Crear un archivo llamado:

HISTORIA.md

Escribir una sola frase.

Hacer un commit.

Modificar la frase.

Hacer otro commit.

Después observar que Git conserva ambas versiones.
